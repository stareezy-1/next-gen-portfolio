Aurora PDF is a privacy-first PDF utility suite that runs entirely in the browser. Core Developer.

## Features

- PDF compression, splitting, and merging
- Watermarking and digital signing
- OCR text extraction via Tesseract.js (WebAssembly)
- Format conversions: Word ↔ PDF, Excel ↔ PDF, PDF → JPG
- Zero server uploads — complete client-side processing
