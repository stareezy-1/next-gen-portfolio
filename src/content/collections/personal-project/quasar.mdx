Quasar is an all-in-one, privacy-first data and code tools suite, part of the Stareezy ecosystem. Core Developer.

## Features

- JSON, XML, CSV, and SQL formatting, validation, and conversion
- 23 color converters (HEX, RGB, HSL, HSV, CMYK, Colortone) and unit converters
- String utilities: case conversion, word counting, encoding, transforms
- Semantic JSON diff and line-by-line text diff
- localStorage sessions with auto-save and named saves
- Registry-driven routing, fuzzy search, and SEO — zero server, zero uploads
