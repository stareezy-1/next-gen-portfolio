Lyra is a payment-native form builder for the Indonesian market, built and operated solo as a personal SaaS. Founder and sole engineer.

## What it does

- Drag-to-build forms with a **payment field** as a first-class block
- **Inline payment** — Virtual Account and QRIS rendered in Lyra's own UI via DOKU SNAP Direct (no hosted-checkout redirect)
- Live **countdown + auto-confirming** pending-payment experience, driven by DOKU webhooks
- **Creator wallet** with transaction history, plus on-demand **withdrawals** to any Indonesian bank
- **Subscriptions** (Free + Pro) and a **referral system** that credits commissions to the wallet
- Email + WhatsApp notifications, PDF receipts, and a PWA install experience

## Stack

Next.js (OpenNext on Cloudflare) · Hono on Cloudflare Workers · D1 + Drizzle · KV · R2 · Queues · Better Auth + Google OAuth · DOKU SNAP payments and Kirim DOKU payouts.
